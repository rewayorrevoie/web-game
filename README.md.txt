# AI Quest Explorer

## Описание игры
Это простая веб-игра в жанре текстовой приключенческой RPG. Игрок берёт роль героя, который исследует фэнтезийный мир, перемещаясь между локациями (лес, гора, пещера, река, замок). В каждой локации может быть монстр для боя, артефакт для сбора или NPC с диалогом. Цель — собрать 3 артефакта, не потеряв все HP (здоровье). Игра использует HTML5 Canvas для визуализации: фоны локаций, иконка персонажа, индикаторы HP и артефактов. Лог событий показывает описания и результаты действий.

- Стартовый экран: Ввод имени и кнопка старта.
- Игровой процесс: Кнопки для движения, атаки, сбора. Рандомные события в локациях.
- Экран завершения: Сообщение о победе/поражении и перезапуск.

## Используемые технологии
- HTML5: Структура страниц и элементов.
- CSS: Стили для интерфейса (центрирование, цвета, размеры).
- Vanilla JavaScript: Логика игры, обработка событий, рисование на Canvas.
- HTML5 Canvas: Для рендеринга визуалов (фоны, иконки, текст).
- Нет внешних фреймворков — всё чистое, чтобы было просто.

## Использование нейросетей
Нейросети использованы в 4 аспектах разработки для автоматизации креативных и технических задач. Инструменты: Grok (xAI) для идей, сценария и кода; DALL·E для изображений.

- **Генерация идеи и концепции**: Инструмент — Grok. Промпт: "Предложи простую веб-игру для браузера на Vanilla JS с элементами приключений, где игрок исследует мир, сражается и собирает предметы. Учти требования: стартовый экран, игровой процесс, экран завершения; используй Canvas для visuals." Результат: Основной сюжет, структура и элементы игры. Сделано с AI: 100%. Вручную: Ничего.
- **Генерация игрового сценария и диалогов**: Инструмент — Grok. Промпт: "Сгенерируй 5 фэнтезийных локаций для приключенческой игры: каждая с описанием, возможным монстром, артефактом и диалогом NPC (если есть). Формат: массив объектов JS." Результат: Массив locations в скрипте с описаниями и диалогами. Сделано с AI: 100%. Вручную: Интеграция в код.
- **Генерация изображений**: Инструмент — DALL·E. Промпты: "Создай пиксель-арт [описание] в фэнтезийном стиле, размер [указан]." Примеры: для player.png — "воина-героя, 64x64, прозрачный фон"; для les.png — "тёмный лес, 400x300". Результат: 6 PNG-файлов в assets/. Сделано с AI: Генерация. Вручную: Скачивание и размещение.
- **Генерация или помощь в написании кода**: Инструмент — Grok. Промпт: "Напиши полный код на Vanilla JS для текстовой RPG-игры на HTML5 Canvas: [детали как в шаге 2]." Результат: Весь <script> в index.html, включая логику. Сделано с AI: 90% (структура и функции). Вручную: Корректировка имён файлов, добавление комментариев.

## Дополнительные задания (реализованные частично)
- AI-управляемый противник: Монстры в локациях — простая рандомная логика (в fight()).
- Сгенерированные уровни: Локации на основе текста от AI.
- Нет адаптивной сложности или достижений — можно добавить вручную (например, увеличить урон монстров с уровнем).

## Запуск игры
1. Клонируйте репозиторий: `git clone https://github.com/yourusername/yourrepo.git`.
2. Откройте index.html в любом браузере.
3. Или онлайн: [Ссылка на GitHub Pages](https://yourusername.github.io/yourrepo/) — после деплоя.

Если баги: Проверьте консоль браузера. Для улучшений — добавьте звуки или больше локаций.
